# Security Policy

## Reporting a Vulnerability

If you believe you've found a security vulnerability in this project, please report it via email:

**Email:** [MAngkad.BSDSBA2027@aim.edu](mailto:MAngkad.BSDSBA2027@aim.edu)

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

I will respond as quickly as possible and work with you to address the issue.
